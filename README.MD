### A simple python module that create a text based image based on the given image.

```console
python main.py \
    -MF "with_ascii/images/imports/saturn-07.jpg" \
    -C 200 \
    -S 70 \
    -OF "with_ascii/images/exports/saturn-07.txt" \
    -BW n
```
